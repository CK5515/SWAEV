Product

Interpretable genomic analysis API

SWAEV Genomics is building a cloud based API that lets researchers and biotech companies to analyse DNA sequences with models that explain why they make predictions. Unlike "black box models", our architecture reveals the biological features (e.g. helical periodicity, regulatory motifs) driving each result.

Key use cases:

  • Regulatory element discovery
  • Variant effect prediction
  • Synthetic promoter design
  • Genome scale functional annotation

We are currently seeking early access partners to validate our platform on proprietary datasets.

Technology

Wave based neural architecture

Our platform extends the open source SPECTRE‑Wave architecture with novel extensions designed specifically for DNA (vocabulary size = 4). The model replaces attention with wave propagation, achieving O(n log n) complexity while maintaining full interpretability.

We have validated the architecture on E. coli, yeast, and human chromosome 22. The model autonomously discovered DNA's helical periodicity (10.5bp), a result that was not programmed in.

Our prototype hardware (AMD RX 6500 XT) limits us to 4 layers and 1024 field size. To scale to full human genomes we need NVIDIA A100/H100 GPUs with CUDA and cuFFT optimisation.
